# Web Fullstack Practice

本仓库用于记录 Web 全栈实习学习过程，包含前端基础、Vue、小程序、Node.js 后端及前后端联调练习。

## 目录结构

- FRONTEND/1_HTML：HTML 基础练习
- FRONTEND/2_CSS3：CSS3 布局与样式练习
- FRONTEND/3_JS：JavaScript 基础与 DOM 操作
- FRONTEND/4_Vue：Vue 3 组件、路由、状态管理、接口请求
- FRONTEND/5_MiniProgram：小程序基础练习
- BACKEND：Node.js 后端与接口开发练习

## 学习目标

1. 熟悉 HTML/CSS/JavaScript 基础
2. 掌握 Vue 3 前端工程化开发
3. 理解小程序基本开发流程
4. 掌握 Node.js 后端接口开发
5. 完成前后端分离项目联调
